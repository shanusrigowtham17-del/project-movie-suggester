Context & Data Scope

  • The Forum: TV & MOVIES community from 7cups.
  • Data Tracked: Thread titles, comfort watch lists, movie quotes, and "Netflix Roulette" social games.
  • Target Purpose: Identifying how users use specific films to soothe stress or numbness.

Tools & Scraping Process

  • Core Tools: Built with Python + Playwright to load heavy JavaScript, and BeautifulSoup to parse text.AI used for development
  • The Hurdle: Dynamic layout code shifts required using flexible HTML relative tree search elements.
  • Safety Limits: Multi-second delays are built into scripts to prevent overloading site servers.

NLP

  • Valence Models: Uses custom VADER NLP classifiers to score post emotion scales from -1.00 (sad) to +1.00 (happy).
  • Topic Clustering: Deploys unsupervised LDA Models to group threads automatically by healing intents.
  • Final Target: Mapping user inputs onto an interactive quadrant grid to suggest movies.

⚖️ Observations

  • Cultural Norms: Members pro-actively swap content trigger warnings before giving reviews.
  • Privacy Safeguards: Scraping remains completely restricted to non-logged-in public zones.
  • Data Purge: User handles are converted to hashes instantly; quotes are slightly tweaked to block reverse search loops.
